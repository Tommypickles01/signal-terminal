Cloud sync (Supabase) — optional

How to turn on cloud sync (free):
1. Make a free project at supabase.com → Project Settings → API → copy the Project URL and anon public key into the fields above.
2. In the Supabase SQL editor, paste & run this:
create table if not exists app_state ( user_id uuid primary key references auth.users(id) on delete cascade, data jsonb not null default '{}'::jsonb, updated_at timestamptz not null default now() ); alter table app_state enable row level security; create policy "own row" on app_state for all using (auth.uid() = user_id) with check (auth.uid() = user_id);
3. Put your email above, hit SEND MAGIC LINK, click the link in your inbox, come back and hit SYNC NOW. Your account, trades, and signal memory now follow you across devices. The anon key is safe to keep in this public file — Row-Level Security means each user only ever sees their own row.
